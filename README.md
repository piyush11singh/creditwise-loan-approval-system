# Cash Me! Maybe? — Credit-Wise Loan Approval System

## Overview
CreditWise is a machine learning project designed to predict loan approval status based on applicant financial and personal information.

The project includes data preprocessing, feature analysis, model training, and prediction workflows using Python and Jupyter Notebook.

---

## Features
- Data Cleaning and Preprocessing
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Loan Approval Prediction
- Machine Learning Model Training
- Model Evaluation

---

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Dataset Features
The dataset contains:
- Applicant Income
- Coapplicant Income
- Loan Amount
- Credit History
- Employment Status
- Education
- Property Area
- Loan Status

---

## Machine Learning Workflow
1. Data Collection
2. Data Cleaning
3. Feature Encoding
4. Model Training
5. Prediction and Evaluation

---

## Goal
To build a machine learning model capable of predicting whether a loan application should be approved or rejected.
